# utils
> 存放一些工具函数

# 函数说明
- plot_scatter_fig.py
    - 将高维数据降维，以散点图显示
- measure.py
    - 多分类数据结果评估
- money_format.py
    - 将汉字金额或者阿拉伯金额转化为统一格式
- edit_distance.py
    - 计算字符串编辑距离
    - 其他实现
        - [fuzzywuzzy](https://github.com/seatgeek/fuzzywuzzy)
        - [python-Levenshtein](https://github.com/miohtama/python-Levenshtein)
